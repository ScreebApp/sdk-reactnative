# screeb-module

A react-native module to integrate Screeb mobile sdk for Android and/or iOS.

## Installation

```sh
npm install screeb-module
```

## Usage

```js
import { multiply } from "screeb-module";

// ...

const result = await multiply(3, 7);
```
